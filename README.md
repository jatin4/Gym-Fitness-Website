HTML and CSS are the basic building blocks used to create websites.

🌐 HTML (HyperText Markup Language)

HTML is used to structure the content of a webpage.

Think of it like the skeleton of a website.

It defines things like:
Headings
Paragraphs
Images
Links
Buttons

Example:

<h1>Hello World</h1>
<p>This is a paragraph.</p>
🎨 CSS (Cascading Style Sheets)

CSS is used to style and design the webpage.

Think of it like the clothes and colors of the website.

It controls things like:
Colors
Fonts
Spacing
Layout
Animations

Example:

h1 {
  color: blue;
  font-size: 24px;
}
🧩 How they work together
HTML = structure (what is on the page)
CSS = style (how it looks)
